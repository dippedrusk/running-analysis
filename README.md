# Running-analysis
# CMPT 318 Final Project

## Get up and running quickly

Download all files and run
```
python3 running_analysis.py GPSdata.csv
```

## Dependencies

* Python (version 3.4)
* NumPy (version 1.13)
* Pandas (version 0.20.3)
* SciPy (version 0.19.1)
* MatPlotLib (version 2.0.2)
* Seaborn (version 0.8.0)
